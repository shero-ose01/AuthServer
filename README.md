# AuthServer

A small REST API for user authentication and authorization, built with ASP.NET Core (controller-based API). Personal learning project to learn more about authentication and autherization with .NET.

## Running this locally:
```bash
git clone https://github.com/shero-ose01/AuthServer.git
cd AuthServer
dotnet restore
dotnet ef database update   # if migrations are used
dotnet run
```
